# Rúbrica de evaluación — Taller MVC (lista de tareas)

Criterios (total 10 pts)
- Funcionalidad básica (4 pts)
  - Crear/leer/editar/eliminar tareas (1 pt cada uno)
- Calidad del código (2 pts)
  - Uso correcto de modelos, vistas y formularios (1 pt)
  - Código limpio y legible (1 pt)
- Interfaz y UX (2 pts)
  - Plantillas coherentes y navegación (1 pt)
  - Mensajes/validación (1 pt)
- Extensiones implementadas (2 pts)
  - Búsqueda, paginación, autenticación o tests (0.5-2 pts según complejidad)

Checklist de entrega
- [ ] Repo con el proyecto o carpeta zip
- [ ] README con instrucciones de ejecución
- [ ] Archivo `requirements.txt`
- [ ] Migraciones aplicadas (o instrucciones para hacerlo)
- [ ] Capturas o enlace a la demo en vivo (opcional)

Comentarios finales
- Puntuar tanto la funcionalidad como la claridad del código y la documentación entregada.
